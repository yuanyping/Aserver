文件管理

a、在上传过程中，尤其是大文件上传需要等待页面下载地址、文件md5值产生才表示上传完成
b、如果同一天同一个项目两个帐号上传相同名字的文件，最终以最后一个帐号上传为准
c、程序最开始是上传文件后会自动计算md5值，对于大文件上传容易导致内存异常，故取消自动计算md5值 d、更新文件结果列表只显示最近10条记录 e、管理员可以进行条件筛选查看所有上传文件列表 f、只有管理员可以看所有的上传结果，非管理员只能查看自己上传的文件
